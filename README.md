# URL Attack Detection System

A cybersecurity project that detects malicious URLs and analyzes network traffic using Machine Learning and PCAP analysis.

## Tech Stack
React
Flask
Python
Machine Learning
Firebase Authentication

## Features
• URL attack detection
• PCAP traffic analysis
• AI explanation of attacks
• Dashboard with attack reports
• Export attack logs

## How to Run

Backend:
pip install -r requirements.txt
python main.py

Frontend:
cd frontend
npm install
npm start

Backend structure added for URL attack detection.
Implemented PCAP file upload and analysis module.
